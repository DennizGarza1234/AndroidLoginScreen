# Login Screen App (Android - ConstraintLayout + Material Design)

## Overview
This project is a professional-grade login screen built using Android Studio.  
It demonstrates responsive UI design using ConstraintLayout and Material Design components.

## Features
- Responsive login screen layout
- Material Design TextInput fields
- Email input with proper keyboard type
- Password field with visibility toggle
- Styled "Sign In" button
- Bottom navigation footer ("Register Now")
- Clean and modern UI design

## Technologies Used
- Kotlin
- Android Studio
- ConstraintLayout
- Material Design Components

## How to Run
1. Clone the repository:
   ```
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
   ```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on an emulator or physical device

## Test Cases

### Normal Test Cases
1. Enter valid email → user@example.com
2. Enter valid password → 12345678
3. Click Sign In button → UI responds correctly

### Edge Test Cases
1. Empty email + password → no crash, empty input handled
2. Invalid email format → UI still accepts input (no crash)
3. Very long text input → layout remains stable and does not break
